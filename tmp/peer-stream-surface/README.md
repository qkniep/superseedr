# Volatility Surface Prototype

Scratch Ratatui prototype for testing a 3D Braille canvas surface before wiring any code into Superseedr.

Run:

```sh
cargo run
```

Controls:

- `h` / `l`: rotate left/right
- `j` / `k`: tilt down/up
- `z` / `x`: zoom in/out
- `space`: pause synthetic data
- `ctrl-r`: reset data
- `q`: quit
